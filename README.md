# SNMP Monitoring & Alert Configuration 

## Overview
Configured an SNMP manager within a simulated enterprise environment to monitor device performance and implement threshold-based alerts for proactive issue detection.

## Key Implementations
- Configured SNMP communication between network devices and manager
- Established alert thresholds for bandwidth utilization and interface status
- Simulated abnormal conditions to validate alert triggers
- Reviewed performance metrics and logs to support troubleshooting

## Skills Demonstrated
- Network monitoring
- SNMP configuration
- Alert threshold tuning
- Proactive incident detection
- Operational troubleshooting
